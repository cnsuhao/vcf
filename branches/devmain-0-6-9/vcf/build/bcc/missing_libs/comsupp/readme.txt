This is partial implementation of comsupp.lib that is distributed with MSVC. 
It is only intended for basic use of MS COM classes like _bstr_t, _variant_t and comptr templates.

The code is designed according to comutil.h/comip.h shipped with Borland Free compiler. 
It has been tested with Borland Free compiler and bcb6.
